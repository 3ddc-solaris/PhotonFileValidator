To achieve optimum results, the following settings should be made in PrusaSlicer:

Please note that the manual applies to PrusaSlicer version 2.2.0. The procedure for other versions may differ.
  • Go to tab "Printer Settings".
  • Select the built-in Prusa SL1 printer profile.
  • Switch the view to expert mode (red button in the upper right corner).
  • In the section "Corrections" set the field "Printer gamma correction" to 0.
    This disables anti-aliasing in PrusaSlicer, as the anti-aliasing settings cannot be applied at the moment. However, this is not a limitation, since the built-in calculation in the PhotonFileValidator offers much finer control of the anti-aliasing settings.
  • (Optional) In the section "Print Host upload" set the field "Hostname, IP or URL" to {URL}
    This setting allows you to send the file to PhotonFileValidator with one click after slicing without having to save and reopen it.
    {WARNING}
  • Save the profile under a new name, for example "Anycubic Photon".
